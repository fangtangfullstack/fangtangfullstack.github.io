# README

最近我们发布了方糖全栈课，整个课程做了四个多月，总共120多节课，课程设计的建议学习时间在15周左右。这是一门以视频为主的课程，发布在网易云课堂上。

这个课程受到了很多同学的欢迎。短短两天就卖出了几十套，而且几乎全是五星的评价。能切实的帮到大家，这让我们觉得辛苦这么久，还是非常的值得的。

另一方面呢，我们也收到了很多同学的反馈。觉得课程虽然很划算，但价格还是超出了他们能承受到范围。（ 原话是，就算使用花呗分期，也超过限额了 😅 ）

所以我们在想，能不能在免费和收费之间，找到一个平衡点。最终我们决定将方糖全栈课的学习路线图+各个章节内容的参考书目和资料，免费发布出来供大家参考。

说实话，在发布之前呢，我也有一些担心，会不会有些同学看了路线图以后就不买视频课程了呢？但最后我们还是决定发布出来。

如果你看完觉得这个路线图帮到了你，请推荐给身边需要这个课程的同学；当然，当你经济允许的时候，购买我们的视频课程就最好了 🤠

同时，视频课程本身虽然更易观看和吸收，但在学完以后，进行资料查找的时候并不是很方便，所以即使对于购买了视频课程的同学，这个路线图和参考资料也会很有价值。

我们会定期更新路线图和参考资料，你可以用微信扫码订阅（未关注关注即可）我们的更新通知。
