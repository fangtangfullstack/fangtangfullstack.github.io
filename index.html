# 课程特色

不着急的同学可以花几分钟听我说说做这个课的起因和心得。

我之所以愿意花这么长时间去做这么一门课，是因为我觉得它对学员的意义和其他课程不一样。它不光是关于技能的，更是关于职业跃迁、职业自由度的。

我最开始是打算做PHP课的，但当我从招聘网站抓取了10万份技术简历（所有技术职位哈）分析后发现，PHP职位和前端职位的技能要求重合度超过了一半。

于是我就想，能不能做一套课程，同时把PHP和前端的内容都覆盖了，这样学员们在选择职业时，职业路线图就会一下多出来一大堆。

比如说，有很多大厂是用Java的，那如果光会PHP，就很难有机会进去。但几乎所有大厂都招前端，如果你前端能力也很不错的话，很可能就入职前端岗了。

另一方面，在二线城市，专职前端的岗位数量可能就会比PHP少很多，PHP会更好找工作。所以只会前端的同学，在这时候，就没有会PHP的同学受欢迎。

顺着这个思路下去，我根据分析数据，精心调整了课程的技术栈，最后惊喜的发现区块链应用开发也能放到这个技术栈里边来，用JS发行加密货币其实很简单。

于是有了现在这个版本的方糖全栈课。

你不用花几倍的钱去买多套课程，只需要花上15周的时间，就可以学会一个面向PHP、前端和区块链Dapp开发三个热门职位的技术栈。

学习没有什么捷径，但是可以有策略，可以有哪些先学、哪些后学。花最少的时间，把使用最多的技能点学会，然后再慢慢补其他知识，可以让你更具备竞争力。

# 课程细节

当然课程好不好我说了也不算，现在我只敢说，像咱这么用心做课的不多，说几个细节。

  • 为了让大家每周能从上下班、通勤路上挤出时间来学习，我们在代码部分录制时开启了放大镜模式，手机上也看得倍儿清楚

  • 为了保证学员在三个操作系统上调试代码环境一致，我们开发了基于docker的 fangPHP 环境来保持一致

  • 为了演示web应用的演化，我们用 原生PHP、自有框架、Bootstrap、React（SPA）、和 React Native 把方糖简历这个项目完整的重写了五遍

  • 为了保持风格统一，课里的绝大部分流程图和图标都是我们自己绘制/重绘（有买授权）的。
